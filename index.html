What is Ghost Trace Echo?

Jennyrave Ghost Trace Echo is a stereo delay plugin designed for the modern producer who demands both surgical precision and analog character. Unlike standard digital delays that sound sterile, or vintage emulations that are too muddy, Ghost Trace sits in the "Modern Lab" sweet spot.

It combines a crystal-clear stereo delay engine with a feedback loop that features "Tanh" saturation - a soft-clipping distortion that adds warmth and grit only to the repeats, keeping your original signal pristine.

Why is Ghost Trace Different?

Inside-the-Loop Processing

Many delays apply EQ or saturation after the delay. Ghost Trace applies it inside the feedback loop. This means the first repeat is slightly colored, the second is more colored, and the tenth is gritty and dark. This creates a natural, organic decay that mimics how sound behaves in the real world.

Signal Flow Diagram

Understanding the signal flow is key to mastering the Ghost Trace Echo. Unlike standard delays, the tone shaping happens inside the feedback loop.

Input -> [Delay Line] -> [Filter (Lo/Hi Cut)] -> [Saturation (Drive)] -> [Feedback Gain] -> Output

Platform: Windows 10/11

To install Jennyrave Ghost Trace Echo, simply place the .vst3 file into your system's dedicated plugin folder.

Common Installation Paths

  • Default System Folder: C:\Program Files\Common Files\VST3
  • Custom Folder: If you use a custom VST folder for your DAW (like Ableton Live or FL Studio), you can place the file there, but the system folder above is the industry standard.

After moving the file, restart your DAW or click "Rescan Plugins" in your preferences.

Control Breakdown

The Ghost Trace interface is divided into three logical panels, following the signal flow from left to right.

1. The Time Panel (Left)

  • SYNC (LCD): Selects the rhythmic interval of the delay (1/32 to 1/1). Includes Triplet (t) and Dotted (.) modifiers.
  • MODE (LCD):
    Single: Standard mono delay.
    Dual: Offset timing for stereo width.
    Ping-Pong: Bounces repeats between left and right.
  • FDBK (Feedback): Controls repeats. 100% creates infinite loops.

2. The Tone Panel (Center)

Controls strictly affect the wet signal (echoes).

  • LO-CUT: High-Pass filter to remove mud.
  • DRIVE: Introduces soft-clipping saturation. High settings create gritty textures that degrade with every repeat.
  • HI-CUT: Low-Pass filter to make echoes sound darker and more distant.

3. The Output Panel (Right)

  • DRY/WET: The mix blend. 100% is essential for Send/Return setups.
  • OUT (Fader): Master volume fader for gain-staging.
  • LVL (Meter): Visual VU meter showing real-time signal strength.
